WorldManager Class
This script runs the entire CivGrid system.

Holds all chunks, and in turn each hexagon, in memory and runs all the operations throughout them when needed. Contains the methods to generate worlds, load worlds, and save worlds. While some generation methods are exposed for use, it is best to not try and use the lower level methods.
