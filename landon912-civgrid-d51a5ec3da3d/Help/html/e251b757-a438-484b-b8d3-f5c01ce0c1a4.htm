Remarks
If openTextureToView is true, it will open the saved file in your default image editor/browser.
Examples
The following will save the texture to a file named savedTexture in the path provided, in this case the root folder of the game, and open the file for viewing.
Copy
class SaveTexture : MonoBehaviour
{
    public Texture2D textureToSave;

    void Start()
    {
        CivGridFileUtility.SaveTexture(textureToSave, Application.dataPath, "savedTexture", true);
    }
}
