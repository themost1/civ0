Examples
The following code saves the terrain to a file called terrainSave.

class SaveTerrain : MonoBehaviour
{

   {
       CivGridFileUtility.SaveTerrain("terrainSave");
   }
}
Remarks
WorldManager must be in the scene and currently have a world in memory. Method will return errors if a WorldManager is not currently in the scene or if it doesn't have a world currently loaded into memory.
