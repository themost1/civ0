HexInfoChangeTextureToResource Method
Switches this hexagon's UV data to display it's resource texture.

Namespace: CivGrid
Assembly: Assembly-CSharp (in Assembly-CSharp.dll) Version: 0.0.0.0
Syntax
C#

public void ChangeTextureToResource()
Examples
The following code changes this hexagon to show its resource texture if a resource and a resource tile texture is present.
Copy
using System;
using UnityEngine;
using CivGrid;

public class ExampleClass : MonoBehaviour
{
  WorldManager worldManager;

   void Start()
   {
       worldManager = GameObject.FindObjectOfType<WorldManager>();

       worldManager.hexChunks[0, 0].hexArray[0, 0].ChangeTextureToResource();
   }
}
